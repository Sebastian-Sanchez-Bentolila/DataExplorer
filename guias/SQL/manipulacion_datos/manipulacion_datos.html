Manipulación de Datos en SQL

SQL proporciona varias instrucciones para manipular datos dentro de las bases de datos. Estas operaciones incluyen la inserción, actualización, eliminación y consulta de datos. A continuación, exploraremos los comandos más utilizados.

INSERT - Insertar Datos

El comando INSERT permite agregar nuevas filas a una tabla.


INSERT INTO empleados (id, nombre, edad, salario)
VALUES (1, 'Carlos Pérez', 30, 50000);

Salida en consola: Se inserta una nueva fila en la tabla empleados.

UPDATE - Actualizar Datos

El comando UPDATE permite modificar registros existentes.


UPDATE empleados
SET salario = 62000
WHERE id = 3;

También podemos actualizar múltiples campos:

Salida en consola: Se actualiza el salario de la empleada con ID 3.

DELETE - Eliminar Datos

El comando DELETE se usa para eliminar registros de una tabla.

DELETE FROM empleados WHERE id = 1;

Salida en consola: Se elimina la fila con ID 1 de la tabla empleados.

Si queremos eliminar todos los registros de una tabla sin eliminar la estructura:

DELETE FROM empleados;

SELECT - Consultar Datos

El comando SELECT permite recuperar información de una tabla.

SELECT * FROM empleados;

Podemos filtrar los resultados:

SELECT nombre, salario FROM empleados WHERE salario > 55000;

Salida en consola: Se muestran todos los registros de la tabla empleados.

WHERE

El operador WHERE se usa para filtrar registros en una consulta.

SELECT * FROM empleados WHERE edad>30;

BETWEEN

Se usa para seleccionar valores dentro de un rango.

SELECT * FROM productos WHERE precio BETWEEN 100 AND 500;

LIKE NOT LIKE

Se usan para buscar patrones en texto.


SELECT * FROM clientes WHERE nombre LIKE 'A%';  -- Nombres que empiezan con 'A'
SELECT * FROM clientes WHERE nombre NOT LIKE '%z';  -- Nombres que no terminan en 'z'

IS NULL e IS NOT NULL

Se usan para filtrar registros que tienen valores nulos o no nulos.


SELECT * FROM empleados WHERE telefono IS NULL;
SELECT * FROM empleados WHERE telefono IS NOT NULL;

DISTINCT

Elimina los valores duplicados de un resultado.

SELECT DISTINCT ciudad FROM clientes;

TRUNCATE - Vaciar una Tabla

El comando TRUNCATE elimina todos los registros de una tabla, pero mantiene su estructura.

TRUNCATE TABLE empleados;

Salida en consola: Se eliminan todos los datos de la tabla empleados.

REPLACE - Inserción o Reemplazo de Datos

El comando REPLACE inserta un nuevo registro o reemplaza uno existente si hay un conflicto de clave primaria.


REPLACE INTO empleados (id, nombre, puesto, salario)
VALUES (1, 'Ana Pérez', 'Gerente', 60000);

Salida en consola: Se reemplaza la fila con ID 1 si ya existía; de lo contrario, se inserta.

LIMIT - Restricción en la Cantidad de Resultados

El comando LIMIT permite restringir la cantidad de registros devueltos por una consulta.


SELECT * FROM empleados
LIMIT 5;

Salida en consola: Se muestran los primeros 5 registros de la tabla empleados.

ORDER BY - Ordenación de Resultados

El comando ORDER BY se usa para ordenar los resultados según una columna específica.


SELECT * FROM empleados
ORDER BY salario DESC;

Salida en consola: Se muestran los empleados ordenados por salario en orden descendente.

GROUP BY - Agrupación de Resultados

El comando GROUP BY permite agrupar resultados según una columna y aplicar funciones agregadas.


SELECT puesto, COUNT(*) AS cantidad
FROM empleados
GROUP BY puesto;

Salida en consola: Se muestra la cantidad de empleados por cada puesto.

HAVING - Filtrado de Agrupaciones

El comando HAVING se usa para filtrar resultados después de agruparlos.


SELECT puesto, COUNT(*) AS cantidad
FROM empleados
GROUP BY puesto
HAVING COUNT(*) > 2;

Salida en consola: Se muestran solo los puestos con más de 2 empleados.

EXISTS - Verificación de Existencia

El comando EXISTS se usa para verificar si una subconsulta devuelve resultados.


SELECT nombre FROM empleados e
WHERE EXISTS (SELECT 1 FROM departamentos d WHERE e.id = d.jefe_id);

Salida en consola: Se listan los nombres de empleados que son jefes de algún departamento.

IN - Búsqueda en una Lista de Valores

El comando IN permite buscar valores dentro de una lista específica.


SELECT * FROM empleados
WHERE puesto IN ('Desarrolladora', 'Gerente');

Salida en consola: Se muestran los empleados que tienen los puestos especificados.

CASE - Condiciones en Consultas

El comando CASE se usa para realizar evaluaciones condicionales dentro de una consulta.


SELECT nombre, salario,
CASE
    WHEN salario > 50000 THEN 'Alto'
    WHEN salario BETWEEN 30000 AND 50000 THEN 'Medio'
    ELSE 'Bajo'
END AS categoria_salarial
FROM empleados;

Salida en consola: Se muestra la categoría salarial de cada empleado.
