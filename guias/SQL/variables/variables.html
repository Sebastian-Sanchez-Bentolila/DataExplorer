📌 Variables en SQL

En SQL, las variables se utilizan para almacenar valores temporales y usarlos dentro de consultas o procedimientos almacenados. Permiten manejar datos de forma flexible dentro del entorno de la base de datos.

🔹 Tipos de Variables en SQL

Existen diferentes tipos de variables según su uso y el sistema de gestión de base de datos (DBMS). Las más comunes son:

  • Variables locales: Se definen dentro de un procedimiento almacenado o función y solo existen en ese contexto.
  • Variables de usuario: Se pueden usar en una sesión de base de datos y existen hasta que finaliza la sesión.
  • Variables de sistema: Son definidas por el DBMS y contienen información sobre la configuración y el estado del sistema.

🔹 Declaración y Uso de Variables Locales

Las variables locales se usan dentro de procedimientos almacenados o funciones.

📌 Sintaxis para Declarar una Variable


DECLARE nombre_variable TIPO_DE_DATO [DEFAULT valor_inicial];

🔹 Ejemplo de Uso en un Procedimiento

📌 Calculamos el salario anual de un empleado usando una variable.


DELIMITER $$

CREATE PROCEDURE CalcularSalarioAnual(IN salario_mensual DECIMAL(10,2))
BEGIN
    DECLARE salario_anual DECIMAL(10,2);
    
    SET salario_anual = salario_mensual * 12;
    
    SELECT salario_anual AS "Salario Anual";
END $$

DELIMITER ;

✅ Llamar al Procedimiento


CALL CalcularSalarioAnual(50000);

💻 Salida en Consola

Salario Anual
600000

🔹 Variables de Usuario

Las variables de usuario se pueden usar en cualquier consulta dentro de una sesión y comienzan con el prefijo @.

📌 Definir y Usar una Variable de Usuario


SET @nombre = 'Sebastián';
SELECT @nombre AS "Nombre del Usuario";

💻 Salida en Consola

Nombre del Usuario
Sebastián

🔹 Uso en Consultas

📌 Usamos una variable para filtrar una consulta.


SET @departamento_id = 101;
SELECT nombre, salario FROM empleados WHERE id_departamento = @departamento_id;

🔹 Variables de Sistema

Las variables de sistema contienen información sobre la configuración y estado del servidor SQL.

📌 Ejemplo de Variables de Sistema


SELECT @@version AS "Versión de MySQL", @@hostname AS "Nombre del Servidor";

💻 Salida en Consola

Versión de MySQL Nombre del Servidor
8.0.33servidor-db

🎯 Diferencias entre Tipos de Variables

Tipo Ámbito Ejemplo de Uso
Variables Locales Solo dentro de un procedimiento o función DECLARE total INT;
Variables de Usuario Disponible en toda la sesión SET @usuario = 'Ana';
Variables de Sistema Configuración del sistema SELECT @@version;
