Clases y Objetos en Python

La programación orientada a objetos (OOP) es un paradigma de programación que utiliza "objetos" y "clases" para organizar el código. En Python, las clases se utilizan para crear objetos, que son instancias de esas clases. Este enfoque permite estructurar programas complejos de una manera más comprensible y modular.

Definición de Clases

Una clase es una plantilla para crear objetos. Define un conjunto de atributos y métodos que los objetos creados a partir de la clase tendrán. A continuación se muestra cómo definir una clase en Python:


# Definir una clase simple llamada "Persona"
class Persona:
    def __init__(self, nombre, edad):
        self.nombre = nombre  # Atributo de instancia
        self.edad = edad      # Atributo de instancia
    
    def saludar(self):
        return f"Hola, mi nombre es {self.nombre} y tengo {self.edad} años."

Creación de Objetos

Un objeto es una instancia de una clase. Para crear un objeto, simplemente llama a la clase como si fuera una función:


# Crear un objeto de la clase Persona
persona1 = Persona("Ana", 30)
persona2 = Persona("Luis", 25)
    
# Acceder a los atributos y métodos del objeto
print(persona1.saludar())
print(persona2.saludar())

Salida en consola:


Hola, mi nombre es Ana y tengo 30 años.
Hola, mi nombre es Luis y tengo 25 años.

Métodos de Clase

Los métodos son funciones definidas dentro de una clase que operan sobre las instancias de la clase. El primer parámetro de cualquier método debe ser self, que hace referencia a la instancia actual de la clase.

Métodos Especiales

Python tiene varios métodos especiales, también conocidos como "métodos mágicos", que comienzan y terminan con dobles guiones bajos. Uno de los métodos especiales más comunes es __init__, que se utiliza para inicializar objetos.


# Definir una clase con métodos especiales
class Punto:
    def __init__(self, x, y):
        self.x = x
        self.y = y
        
    def __str__(self):
        return f"({self.x}, {self.y})"
        
    def __add__(self, otro):
        return Punto(self.x + otro.x, self.y + otro.y)

Uso de Métodos Especiales


# Crear objetos de la clase Punto
punto1 = Punto(1, 2)
punto2 = Punto(3, 4)
    
# Usar métodos especiales
print(punto1)             # Llama a __str__
punto3 = punto1 + punto2  # Llama a __add__
print(punto3)

Salida en consola:


(1, 2)
(4, 6)

Herencia

La herencia permite crear una nueva clase basada en una clase existente. La nueva clase (subclase) hereda los atributos y métodos de la clase base (superclase), y puede agregar nuevos atributos y métodos o sobrescribir los existentes.

Definición de una Subclase


# Definir una clase base
class Animal:
    def __init__(self, nombre):
        self.nombre = nombre
        
    def hablar(self):
        pass
    
# Definir una subclase que hereda de Animal
class Perro(Animal):
    def hablar(self):
        return "Guau!"
    
# Definir otra subclase que hereda de Animal
class Gato(Animal):
    def hablar(self):
        return "Miau!"

Uso de Subclases


# Crear objetos de las subclases
perro = Perro("Rex")
gato = Gato("Miau")
    
# Llamar a métodos sobrescritos
print(perro.nombre, "dice:", perro.hablar())
print(gato.nombre, "dice:", gato.hablar())

Salida en consola:


Rex dice: Guau!
Miau dice: Miau!

Encapsulamiento

El encapsulamiento es el principio de restringir el acceso directo a algunos de los atributos y métodos de un objeto, para evitar modificaciones accidentales o no autorizadas. En Python, esto se logra utilizando prefijos de subrayado:


# Definir una clase con atributos encapsulados
class CuentaBancaria:
    def __init__(self, saldo_inicial):
        self.__saldo = saldo_inicial  # Atributo privado
    
    def depositar(self, monto):
        self.__saldo += monto
    
    def retirar(self, monto):
        if monto <= self.__saldo:
            self.__saldo -= monto
        else:
            print("Fondos insuficientes")
    
    def mostrar_saldo(self):
        return self.__saldo

Uso de Encapsulamiento


# Crear un objeto de la clase CuentaBancaria
cuenta = CuentaBancaria(1000)
    
# Interactuar con métodos públicos
cuenta.depositar(500)
cuenta.retirar(200)
print("Saldo:", cuenta.mostrar_saldo())
    
# Intentar acceder a un atributo privado (generará un error)
print(cuenta.__saldo)

Salida en consola:


Saldo: 1300
AttributeError: 'CuentaBancaria' object has no attribute '__saldo'

Polimorfismo

El polimorfismo permite utilizar una interfaz común para diferentes tipos de objetos. En Python, esto se puede lograr mediante la herencia y la sobrescritura de métodos.

Ejemplo de Polimorfismo


# Definir una función que acepta diferentes tipos de objetos
def hacer_hablar(animal):
    print(animal.hablar())
    
# Crear objetos de diferentes subclases
perro = Perro("Rex")
gato = Gato("Miau")
    
# Usar polimorfismo
hacer_hablar(perro)
hacer_hablar(gato)

Salida en consola:


Guau!
Miau!
