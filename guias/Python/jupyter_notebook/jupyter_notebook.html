Introducción a Jupyter Notebook

Jupyter Notebook es una herramienta interactiva ampliamente utilizada en ciencia de datos, aprendizaje automático y desarrollo de software. Permite combinar código, texto formateado (usando Markdown), gráficos y ecuaciones matemáticas en un único documento ejecutable. Es ideal para la enseñanza, la exploración de datos y la creación de informes interactivos.

Instalación de Jupyter Notebook

Jupyter Notebook se puede instalar de varias maneras. Aquí te mostramos las opciones más comunes:

Opción 1: Instalación usando Anaconda (Recomendado)

  1. Descarga e instala Anaconda, una distribución popular de Python que incluye Jupyter Notebook.
  2. Abre Anaconda Navigator y selecciona "Jupyter Notebook" para iniciarlo.

Opción 2: Instalación con pip

  1. Asegúrate de tener Python instalado. Puedes descargarlo desde python.org.
  2. Abre una terminal o línea de comandos e instala Jupyter Notebook con el siguiente comando: 
    pip install notebook
  3. Inicia Jupyter Notebook escribiendo: 
    jupyter notebook

Opción 3: Instalación en un entorno virtual

Si prefieres trabajar en un entorno virtual para mantener tus dependencias organizadas, sigue estos pasos:

  1. Crea un entorno virtual: 
    python -m venv nombre_entorno
  2. Activa el entorno:
    • En Windows: nombre_entorno\Scripts\activate
    • En macOS/Linux: source nombre_entorno/bin/activate
  3. Instala Jupyter Notebook: 
    pip install notebook
  4. Inicia Jupyter Notebook: 
    jupyter notebook

¿Cómo usar Jupyter Notebook?

Una vez instalado, sigue estos pasos para usar Jupyter Notebook:

  1. Ejecuta el comando jupyter notebook en tu terminal. Se abrirá una interfaz web en tu navegador.
  2. Explora tus directorios desde la interfaz y crea un nuevo notebook haciendo clic en "New" y seleccionando "Python 3".
  3. Escribe tu código o texto en las celdas del notebook. Para ejecutar una celda, presiona Shift + Enter.

Funcionalidades principales

Jupyter Notebook ofrece una variedad de características útiles:

  • Código ejecutable: Escribe y ejecuta código Python en celdas.
  • Texto con Markdown: Formatea texto usando Markdown para agregar títulos, listas, enlaces y más.
  • Gráficos: Integra bibliotecas como Matplotlib o Seaborn para generar gráficos interactivos.
  • Ecuaciones matemáticas: Usa LaTeX para escribir ecuaciones matemáticas.

Atajos útiles

Aprender algunos atajos de teclado puede acelerar tu flujo de trabajo:

Atajo Acción
Shift + Enter Ejecutar celda y pasar a la siguiente
Ctrl + Enter Ejecutar celda sin moverse
Esc + M Convertir celda a Markdown
Esc + Y Convertir celda a código
Esc + A Insertar celda arriba
Esc + B Insertar celda abajo

Ventajas de usar Jupyter Notebook

  • Interactividad: Ideal para explorar y visualizar datos.
  • Versatilidad: Soporta múltiples lenguajes de programación (con extensiones).
  • Compartibilidad: Fácil de compartir como archivo .ipynb o exportar a HTML, PDF, etc.
