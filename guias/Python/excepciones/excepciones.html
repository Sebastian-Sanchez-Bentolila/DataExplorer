Manejo de Excepciones en Python

El manejo de excepciones es una parte crucial de la programación que permite a los desarrolladores gestionar errores de manera controlada. En Python, el manejo de excepciones se realiza principalmente utilizando las sentencias try, except, else, finally y raise. A continuación, exploraremos cada uno de estos elementos con ejemplos prácticos.

La Sentencia try y except

La sentencia try permite probar un bloque de código en busca de errores, mientras que except maneja las excepciones que puedan ocurrir:


try:
    # Bloque de código a probar
    resultado = 10 / 0
except ZeroDivisionError:
    # Manejo de la excepción
    print("Error: División por cero no permitida.")

Salida en consola:


Error: División por cero no permitida.

La Sentencia else

El bloque else se ejecuta si no se produce ninguna excepción en el bloque try:


try:
    resultado = 10 / 2
except ZeroDivisionError:
    print("Error: División por cero no permitida.")
else:
    print(f"El resultado es: {resultado}")

Salida en consola:


El resultado es: 5.0

La Sentencia finally

El bloque finally se ejecuta siempre, ocurra o no una excepción. Se utiliza para liberar recursos o ejecutar código de limpieza:


try:
    resultado = 10 / 2
except ZeroDivisionError:
    print("Error: División por cero no permitida.")
else:
    print(f"El resultado es: {resultado}")
finally:
    print("La ejecución del bloque try-except ha terminado.")

Salida en consola:


El resultado es: 5.0
La ejecución del bloque try-except ha terminado.

La Sentencia raise

La sentencia raise permite generar una excepción en cualquier momento del código. Esto es útil para probar el manejo de excepciones o para gestionar errores específicos:


def dividir(a, b):
    if b == 0:
        raise ValueError("El divisor no puede ser cero.")
    return a / b

try:
    resultado = dividir(10, 0)
except ValueError as e:
    print(f"Error: {e}")

Salida en consola:


Error: El divisor no puede ser cero.

Manejo de Múltiples Excepciones

Es posible manejar múltiples excepciones utilizando varios bloques except o especificando varias excepciones en un solo bloque except:


try:
    resultado = 10 / int("a")
except (ValueError, ZeroDivisionError) as e:
    print(f"Error: {e}")

Salida en consola:


Error: invalid literal for int() with base 10: 'a'

Creación de Excepciones Personalizadas

En Python, también puedes crear tus propias excepciones personalizadas definiendo una nueva clase que herede de Exception:


# Definir una excepción personalizada
class MiExcepcion(Exception):
    pass

def verificar(valor):
    if valor < 0:
        raise MiExcepcion("El valor no puede ser negativo.")
    return valor

try:
    verificar(-1)
except MiExcepcion as e:
    print(f"Error: {e}")

Salida en consola:


Error: El valor no puede ser negativo.
