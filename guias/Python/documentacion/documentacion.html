Documentación en Python

La documentación es una parte esencial del desarrollo de software. Ayuda a los desarrolladores a entender cómo usar tu código y facilita el mantenimiento del mismo. En Python, las cadenas de documentación (docstrings) juegan un papel crucial en la creación de documentación clara y útil.

Docstrings

Los docstrings son cadenas de texto que se colocan al inicio de módulos, clases, métodos o funciones. Sirven para describir su propósito y uso. Aquí tienes un ejemplo básico:


def mi_funcion(param1, param2):
    """
    Esta es una función de ejemplo.
        
    Parámetros:
    param1 (int): El primer parámetro.
    param2 (str): El segundo parámetro.
        
    Retorna:
    bool: El valor de retorno.
    """
    return True

Convenciones de Docstrings

Es importante seguir ciertas convenciones al escribir docstrings para asegurar la claridad y consistencia. Algunos estilos comunes incluyen:

  • Estilo de Google
  • Estilo de NumPy/SciPy
  • Estilo de reStructuredText (reST)

def suma(a, b):
    """
    Suma dos números.
    
    Parámetros
    ----------
    a : int
        El primer número.
    b : int
        El segundo número.
    
    Retorna
    -------
    int
        La suma de los dos números.
    """
    return a + b

Ejemplo Completo

Aquí tienes un ejemplo completo de cómo documentar una clase utilizando docstrings:


"""
Módulo de ejemplo
=================
    
Este es un módulo de ejemplo para demostrar cómo documentar código en Python.
"""
    
class Ejemplo:
    """
    Clase de ejemplo.
    
    Atributos
    ----------
    x : int
        Un valor entero.
    y : int
        Otro valor entero.
    
    Métodos
    -------
    __init__(x, y)
        Inicializa los valores de la clase.
    sumar()
        Retorna la suma de x e y.
    """
    def __init__(self, x, y):
        """
        Inicializa los valores de la clase.
    
        Parámetros
        ----------
        x : int
            Un valor entero.
        y : int
            Otro valor entero.
        """
        self.x = x
        self.y = y
    
    def sumar(self):
        """
        Retorna la suma de x e y.
    
        Retorna
        -------
        int
            La suma de x e y.
        """
        return self.x + self.y

Notas y Advertencias

Nota: Una buena práctica es mantener la documentación actualizada conforme cambias tu código. Documentar el código no solo es útil para otros desarrolladores, sino también para ti en el futuro.
