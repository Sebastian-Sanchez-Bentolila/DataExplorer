Funciones en Python

Las funciones en Python son bloques de código reutilizables que realizan una tarea específica. Permiten estructurar el código de manera modular, facilitando su lectura y mantenimiento. A continuación, exploraremos los conceptos básicos y avanzados de las funciones en Python, junto con ejemplos prácticos.

Definición de una Función

Para definir una función en Python, se utiliza la palabra clave def, seguida del nombre de la función, paréntesis y dos puntos. Dentro de la función, se incluye el bloque de código que realiza la tarea deseada:


# Definir una función simple
def saludar():
    print("¡Hola, mundo!")

Llamada a una Función

Para llamar a una función, simplemente escribe su nombre seguido de paréntesis:


# Llamar a la función saludar
saludar()

Salida en consola:


¡Hola, mundo!

Parámetros y Argumentos

Las funciones pueden aceptar parámetros, que son valores que se pasan a la función cuando se llama. Los parámetros se definen dentro de los paréntesis de la definición de la función:


# Definir una función con parámetros
def saludar(nombre):
    print(f"¡Hola, {nombre}!")

# Llamar a la función con un argumento
saludar("Ana")

Salida en consola:


¡Hola, Ana!

Valores de Retorno

Las funciones pueden devolver valores utilizando la palabra clave return. El valor de retorno puede ser asignado a una variable:


# Definir una función que devuelve un valor
def sumar(a, b):
    return a + b

# Llamar a la función y almacenar el valor de retorno en una variable
resultado = sumar(3, 5)
print(f"La suma es: {resultado}")

Salida en consola:


La suma es: 8

Argumentos por Defecto

Las funciones pueden tener parámetros con valores por defecto, que se utilizan si no se proporciona un valor para el argumento correspondiente al llamar a la función:


# Definir una función con un argumento por defecto
def saludar(nombre="mundo"):
    print(f"¡Hola, {nombre}!")

# Llamar a la función sin argumentos
saludar()

# Llamar a la función con un argumento
saludar("Luis")

Salida en consola:


¡Hola, mundo!
¡Hola, Luis!

Argumentos con Nombre (Keyword Arguments)

Al llamar a una función, puedes especificar los argumentos utilizando el nombre del parámetro. Esto se conoce como argumentos con nombre:


# Definir una función con varios parámetros
def presentar(nombre, edad, ciudad):
    print(f"Soy {nombre}, tengo {edad} años y vivo en {ciudad}.")

# Llamar a la función con argumentos con nombre
presentar(edad=30, ciudad="Buenos Aires", nombre="Ana")

Salida en consola:


Soy Ana, tengo 30 años y vivo en Buenos Aires.

Funciones con Número Variable de Argumentos

Python permite definir funciones que acepten un número variable de argumentos utilizando *args y **kwargs:

*args

El uso de *args permite pasar un número variable de argumentos posicionales a una función:


# Definir una función que acepta un número variable de argumentos posicionales
def sumar(*args):
    return sum(args)

# Llamar a la función con diferentes números de argumentos
print(sumar(1, 2, 3))
print(sumar(4, 5, 6, 7))

Salida en consola:


6
22

**kwargs

El uso de **kwargs permite pasar un número variable de argumentos con nombre a una función:


# Definir una función que acepta un número variable de argumentos con nombre
def mostrar_info(**kwargs):
    for clave, valor in kwargs.items():
        print(f"{clave}: {valor}")

# Llamar a la función con diferentes números de argumentos con nombre
mostrar_info(nombre="Ana", edad=30)
mostrar_info(ciudad="Buenos Aires", país="Argentina", año=2024)

Salida en consola:


nombre: Ana
edad: 30
ciudad: Buenos Aires
país: Argentina
año: 2024

Funciones Lambda

Las funciones lambda, también conocidas como funciones anónimas, son funciones pequeñas y anónimas que se definen utilizando la palabra clave lambda. Son útiles para crear funciones rápidas y sencillas:


# Definir una función lambda que suma dos números
sumar = lambda a, b: a + b

# Llamar a la función lambda
resultado = sumar(3, 5)
print(f"La suma es: {resultado}")

Salida en consola:


La suma es: 8

Funciones Anidadas y Ámbito de Variables

Las funciones pueden definirse dentro de otras funciones. El ámbito de una variable es el contexto en el que se define y es accesible:


# Definir una función externa
def funcion_externa():
    x = 10  # Variable local en el ámbito de la función externa

    # Definir una función interna
    def funcion_interna():
        print(f"El valor de x es: {x}")

    # Llamar a la función interna
    funcion_interna()

# Llamar a la función externa
funcion_externa()

Salida en consola:


El valor de x es: 10

Decoradores de Funciones

Los decoradores son funciones que modifican el comportamiento de otras funciones. Se utilizan para agregar funcionalidad adicional a una función existente:


# Definir un decorador
def mi_decorador(func):
    def nueva_funcion():
        print("Antes de la función")
        func()
        print("Después de la función")
    return nueva_funcion

# Usar el decorador
@mi_decorador
def saludar():
    print("¡Hola!")

# Llamar a la función decorada
saludar()

Salida en consola:


Antes de la función
¡Hola!
Después de la función

Documentación de Funciones

Es importante documentar las funciones para que otros desarrolladores (o tú mismo en el futuro) puedan entender su propósito y uso. Se recomienda utilizar cadenas de documentación (docstrings):


def sumar(a, b):
    """
    Suma dos números y devuelve el resultado.

    Args:
        a (int o float): El primer número.
        b (int o float): El segundo número.

    Returns:
        int o float: La suma de a y b.
    """
    return a + b

# Ver la documentación de la función
print(sumar.__doc__)

Salida en consola:


Suma dos números y devuelve el resultado.

Args:
    a (int o float): El primer número.
    b (int o float): El segundo número.

Returns:
    int o float: La suma de a y b.
