Introducción al Lenguaje Python

Explorando uno de los lenguajes de programación más populares y versátiles del mundo.

¿Qué es Python?

Python es un lenguaje de programación interpretado, de alto nivel y propósito general. Su diseño enfatiza la legibilidad del código, utilizando una sintaxis simple que facilita el aprendizaje y la colaboración.

Es un lenguaje multiparadigma que admite programación orientada a objetos, funcional e imperativa.

Python también destaca por su amplia comunidad y ecosistema de bibliotecas, lo que permite abordar proyectos en áreas como ciencia de datos, inteligencia artificial, desarrollo web, y mucho más.

Origen y Motivación

Python fue creado por Guido van Rossum y se lanzó por primera vez en 1991. Su desarrollo comenzó a finales de los años 80, y fue concebido como una mejora de un lenguaje anterior llamado ABC.

El nombre "Python" no proviene del animal, sino del grupo de comedia británico Monty Python. Esto refleja el enfoque del lenguaje hacia la sencillez y el disfrute.

En comparación con otros lenguajes de su época, como C y Perl, Python ofrecía una sintaxis más limpia y una curva de aprendizaje más suave, lo que lo convirtió en una opción atractiva para principiantes y expertos.

Usos de Python

  • Desarrollo web (frameworks como Django, Flask).
  • Análisis de datos y ciencia de datos (pandas, NumPy, SciPy).
  • Inteligencia artificial y aprendizaje automático (TensorFlow, PyTorch).
  • Automatización y scripting.
  • Desarrollo de videojuegos (Pygame).
  • Aplicaciones de escritorio y GUIs (Tkinter, PyQt).
  • Internet de las cosas (IoT) y hardware.

Entornos de Desarrollo

Python se puede desarrollar en diversos entornos. Algunos de los más populares incluyen:

  • PyCharm: Un IDE profesional desarrollado por JetBrains, ideal para proyectos grandes.
  • Visual Studio Code: Un editor ligero con extensiones potentes para Python.
  • Jupyter Notebook: Ideal para ciencia de datos y aprendizaje automático.
  • Spyder: Orientado a la programación científica.
  • IDLE: El entorno básico incluido con la instalación de Python.

Para quienes recién comienzan, IDLE y Visual Studio Code son excelentes opciones. IDLE es sencillo y directo, mientras que Visual Studio Code ofrece herramientas más avanzadas a medida que se adquiere experiencia.

Tipos de Datos en Python

Python admite diversos tipos de datos fundamentales y estructuras avanzadas:

Tipos Primitivos:

  • int: Números enteros (ejemplo: 10, -5).
  • float: Números decimales (ejemplo: 3.14, -0.5).
  • str: Cadenas de texto (ejemplo: "Hola", "Python").
  • bool: Valores lógicos (True, False).

Estructuras de Datos:

  • list: Listas ordenadas y mutables (ejemplo: [1, 2, 3]).
  • tuple: Tuplas ordenadas e inmutables (ejemplo: (1, 2, 3)).
  • dict: Diccionarios (pares clave-valor) (ejemplo: {"nombre": "Python", "año": 1991}).
  • set: Conjuntos desordenados y únicos (ejemplo: {1, 2, 3}).

Ejemplo de Código:


# Ejemplo de diferentes tipos de datos
entero = 10
flotante = 3.14
cadena = "Hola, Python!"
booleano = True
lista = [1, 2, 3]
tupla = (1, 2, 3)
diccionario = {"clave": "valor"}
conjunto = {1, 2, 3}

print(entero, flotante, cadena, booleano)
print(lista, tupla, diccionario, conjunto)

Salida en consola: 


10 3.14 Hola, Python! True
[1, 2, 3] (1, 2, 3) {'clave': 'valor'} {1, 2, 3}

Operadores en Python

1. Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar operaciones matemáticas básicas. Funcionan con números y devuelven un resultado basado en la operación realizada.

Operador Descripción Ejemplo Resultado
+ Suma 5 + 3 8
- Resta 10 - 4 6
* Multiplicación 6 * 7 42
/ División 8 / 2 4.0
% Módulo (resto) 10 % 3 1
** Exponenciación 2 ** 3 8
// División entera 9 // 2 4

2. Operadores de Comparación

Los operadores de comparación comparan dos valores y devuelven un resultado booleano: True o False.

Operador Descripción Ejemplo Resultado
== Igual a 5 == 5 True
!= Diferente de 5 != 3 True
> Mayor que 7 > 3 True
< Menor que 3 < 5 True
>= Mayor o igual que 5 >= 5 True
<= Menor o igual que 4 <= 6 True

3. Operadores Lógicos

Los operadores lógicos combinan expresiones booleanas y devuelven True o False según las condiciones.

Operador Descripción Ejemplo Resultado
and Devuelve True si ambas condiciones son verdaderas. (5 > 3) and (6 > 2) True
or Devuelve True si al menos una condición es verdadera. (5 > 3) or (2 > 6) True
not Invierte el valor lógico. not (5 > 3) False

4. Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a las variables. Algunos combinan operaciones matemáticas con asignaciones.

Operador Descripción Ejemplo Resultado
= Asigna un valor. x = 5 x es 5
+= Suma y asigna. x += 3 x = x + 3
-= Resta y asigna. x -= 2 x = x - 2
*= Multiplica y asigna. x *= 4 x = x * 4
/= Divide y asigna. x /= 2 x = x / 2
%= Calcula el módulo y asigna. x %= 3 x = x % 3
//= Divide enteros y asigna. x //= 2 x = x // 2
**= Eleva a potencia y asigna. x **= 2 x = x ** 2

Ejemplo práctico


# Operadores aritméticos
a = 10
b = 3
print(a + b)  # Suma
print(a % b)  # Módulo

# Operadores de comparación
print(a > b)  # Mayor que
print(a == b)  # Igualdad

# Operadores lógicos
print((a > b) and (b > 0))  # Lógico AND
print(not (a == b))  # Negación lógica

# Operadores de asignación
a += 5
print(a)  # a ahora es 15

Estructuras de Control de Flujo

Las estructuras de control de flujo permiten dirigir la ejecución del programa según ciertas condiciones o repeticiones.

Condicionales

Los condicionales permiten ejecutar bloques de código según se cumplan ciertas condiciones.


# Ejemplo de condicional
edad = 18
if edad >= 18:
    print("Eres mayor de edad")
elif edad >= 13:
    print("Eres adolescente")
else:
    print("Eres menor de edad")

Salida en consola: 


Eres mayor de edad

Bucles

Los bucles se usan para repetir un bloque de código varias veces:

  • while: Repite mientras una condición sea verdadera.
  • for: Itera sobre una secuencia (lista, rango, etc.).

# Ejemplo de bucle for
for i in range(5):
    print(i)

# Ejemplo de bucle while
contador = 0
while contador < 5:
    print(contador)
    contador += 1

Salida en consola:

1
2
3
4
0
1
2
3
4

Funciones

Las funciones son bloques de código reutilizables que se pueden invocar para realizar tareas específicas.


# Ejemplo de función
def saludar(nombre):
    return f"Hola, {nombre}!"

print(saludar("Python"))

Salida en consola: 


Hola, Python!

Manejo de Excepciones

El manejo de excepciones permite controlar errores durante la ejecución del programa, evitando que este termine abruptamente.


# Ejemplo de manejo de excepciones
try:
    numero = int(input("Ingresa un número: "))
    print(f"El número ingresado es {numero}")
except ValueError:
    print("Eso no es un número válido")
finally:
    print("Fin del programa")

Salida en consola: 


Ingresa un número: t
Eso no es un número válido
Fin del programa

Comentarios en Python

Los comentarios son fragmentos de texto que se incluyen en el código para explicar su propósito o funcionamiento. Python no los ejecuta, lo que los convierte en herramientas ideales para documentación interna.

Comentarios de una sola línea

Para escribir un comentario de una sola línea, utiliza el símbolo #. Todo lo que esté a la derecha de este símbolo será ignorado por Python. 


# Esto es un comentario de una sola línea
print("Hola, mundo")  # Este comentario explica el print

Comentarios de varias líneas

Python no tiene un bloque de comentarios nativo, pero puedes usar cadenas de texto triples (""" o ''') como una alternativa para agregar comentarios largos o descripciones. 


"""
Este es un comentario de varias líneas.
Se puede usar para explicar una sección completa del código.
"""
print("Este código tiene comentarios explicativos.")

Buenas prácticas al usar comentarios

  • Escribe comentarios claros y concisos.
  • Evita comentarios redundantes. Por ejemplo, no expliques algo que ya es obvio en el código.
  • Actualiza los comentarios si cambias el código relacionado.
  • Usa comentarios para describir la lógica o el propósito, no para narrar cada línea.

Identificadores en Python

Un identificador es el nombre que se asigna a variables, funciones, clases o módulos en Python. Siguen ciertas reglas y convenciones específicas que debes respetar.

Reglas para nombres de identificadores

  • Debe comenzar con una letra (a-z, A-Z) o un guion bajo (_).
  • Puede contener letras, dígitos (0-9) y guion bajo (_).
  • No puede comenzar con un dígito.
  • No puede coincidir con palabras clave reservadas de Python como if, else, for, etc.
  • Sensible a mayúsculas y minúsculas: Variable y variable son diferentes.

# Ejemplo válido
nombre_usuario = "Sebastián"
Edad = 19
_estilo_clase = "sutil"

# Ejemplo inválido
2variable = 10  # Comienza con un número
class = "Python"  # Usa una palabra clave reservada

Convenciones de nombres

  • snake_case: Usado comúnmente para variables y funciones (mi_variable).
  • PascalCase: Usado para nombres de clases (MiClase).
  • UPPER_CASE: Usado para constantes (CONSTANTE).
  • Evita usar una sola letra para nombres de variables, excepto en bucles (i, j).

Ejemplo práctico: Comentarios e Identificadores

Observa el siguiente fragmento de código que utiliza comentarios e identificadores siguiendo las mejores prácticas: 


# Este programa calcula el área de un rectángulo
def calcular_area(base, altura):
    """Calcula y retorna el área de un rectángulo"""
    return base * altura

# Variables principales
base_rectangulo = 10
altura_rectangulo = 5

# Cálculo del área
area = calcular_area(base_rectangulo, altura_rectangulo)
print(f"El área del rectángulo es: {area}")

Salida en consola:

El área del rectángulo es: 50
