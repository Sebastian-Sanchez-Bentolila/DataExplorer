Introducción a Python

Python es un lenguaje de programación de alto nivel, fácil de aprender y muy utilizado en diversas áreas como Data Science, Desarrollo Web y Automatización.

Paso 1: Instalar Python

1. Ve al sitio oficial de Python: python.org.
2. Descarga la última versión de Python para tu sistema operativo.
3. Durante la instalación, selecciona la opción "Add Python to PATH" para configurarlo correctamente.

Paso 2: Escribir tu Primer Script

Escribe el siguiente código en un archivo con extensión .py y ejecútalo desde la terminal:

print("¡Hola, Mundo!")

Deberías ver el mensaje: ¡Hola, Mundo!

Paso 3: Variables y Tipos de Datos

Aprende cómo funcionan las variables y los diferentes tipos de datos en Python:

# Números
edad = 25
pi = 3.14159

# Cadenas
nombre = "Sebastián"
mensaje = "¡Bienvenido a Python!"

# Booleanos
es_estudiante = True

print(nombre, "tiene", edad, "años.")

Paso 4: Estructuras de Control

Controla el flujo de tu programa con condicionales y bucles:

# Condicional
numero = 10
if numero > 5:
    print("El número es mayor que 5.")

# Bucle For
for i in range(5):
    print("Iteración:", i)

# Bucle While
contador = 0
while contador < 3:
    print("Contador:", contador)
    contador += 1

Paso 5: Listas y Funciones

Trabaja con listas y define tus propias funciones:

# Listas
frutas = ["Manzana", "Banana", "Cereza"]
for fruta in frutas:
    print("Me gusta:", fruta)

# Funciones
def saludo(nombre):
    return f"Hola, {nombre}"

print(saludo("Sebastián"))

Notas Finales

  • Python tiene una amplia comunidad y bibliotecas que facilitan el desarrollo.
  • Practica resolviendo problemas básicos y explorando nuevas funcionalidades.
  • Usa recursos como W3Schools o Real Python para seguir aprendiendo.
