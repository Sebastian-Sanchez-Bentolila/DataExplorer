Expresiones Regulares en Python

Las expresiones regulares son una herramienta poderosa para trabajar con texto. Permiten buscar, coincidir y manipular patrones específicos en cadenas de caracteres. En Python, el módulo re proporciona soporte para expresiones regulares. A continuación, exploraremos los conceptos básicos de expresiones regulares y cómo usarlas en Python.

Instalación y Uso del Módulo re

El módulo re está incluido en la biblioteca estándar de Python, por lo que no es necesario instalarlo. Puedes importarlo y comenzar a usarlo directamente:


import re

Funciones Principales del Módulo re

El módulo re ofrece varias funciones para trabajar con expresiones regulares:

  • re.match(): Comprueba si el patrón coincide con el comienzo de la cadena.
  • re.search(): Busca el primer lugar donde el patrón coincide con la cadena.
  • re.findall(): Devuelve una lista de todas las coincidencias del patrón en la cadena.
  • re.finditer(): Devuelve un iterador con todas las coincidencias del patrón en la cadena.
  • re.sub(): Reemplaza las coincidencias del patrón en la cadena por un texto de reemplazo.

Ejemplos Prácticos

Coincidencia de Patrones

El siguiente ejemplo muestra cómo usar re.match() para comprobar si una cadena comienza con un patrón específico:


import re
    
# Patrón y cadena
patron = r"hola"
cadena = "hola mundo"
    
# Comprobar coincidencia al inicio de la cadena
coincidencia = re.match(patron, cadena)
    
if coincidencia:
    print("¡Coincidencia encontrada!")
else:
    print("No se encontró coincidencia.")

Salida en consola:


¡Coincidencia encontrada!

Búsqueda en una Cadena

El siguiente ejemplo muestra cómo usar re.search() para buscar la primera coincidencia de un patrón en una cadena:


import re
    
# Patrón y cadena
patron = r"mundo"
cadena = "hola mundo"
    
# Buscar primera coincidencia en la cadena
coincidencia = re.search(patron, cadena)
    
if coincidencia:
    print("¡Coincidencia encontrada!")
else:
    print("No se encontró coincidencia.")

Salida en consola:


¡Coincidencia encontrada!

Encontrar Todas las Coincidencias

El siguiente ejemplo muestra cómo usar re.findall() para encontrar todas las coincidencias de un patrón en una cadena:


import re
    
# Patrón y cadena
patron = r"\d+"
cadena = "Los números son 123, 456 y 789."
    
# Encontrar todas las coincidencias en la cadena
coincidencias = re.findall(patron, cadena)
    
print("Coincidencias encontradas:", coincidencias)

Salida en consola:


Coincidencias encontradas: ['123', '456', '789']

Iterar sobre Coincidencias

El siguiente ejemplo muestra cómo usar re.finditer() para iterar sobre todas las coincidencias de un patrón en una cadena:


import re
    
# Patrón y cadena
patron = r"\b\w+\b"
cadena = "Este es un ejemplo."
    
# Iterar sobre todas las coincidencias en la cadena
for coincidencia in re.finditer(patron, cadena):
    print("Coincidencia encontrada:", coincidencia.group())

Salida en consola:


Coincidencia encontrada: Este
Coincidencia encontrada: es
Coincidencia encontrada: un
Coincidencia encontrada: ejemplo

Reemplazo de Patrones

El siguiente ejemplo muestra cómo usar re.sub() para reemplazar todas las coincidencias de un patrón en una cadena:


import re
    
# Patrón y cadena
patron = r"\d+"
cadena = "Los números son 123, 456 y 789."
reemplazo = "#"
    
# Reemplazar todas las coincidencias en la cadena
resultado = re.sub(patron, reemplazo, cadena)
    
print("Resultado:", resultado)

Salida en consola:


Resultado: Los números son #, # y #.

Expresiones Regulares Comunes

A continuación, se muestran algunos patrones de expresiones regulares comunes y su uso:

  • \d: Coincide con cualquier dígito.
  • \D: Coincide con cualquier carácter que no sea un dígito.
  • \w: Coincide con cualquier carácter alfanumérico (letra o número).
  • \W: Coincide con cualquier carácter no alfanumérico.
  • \s: Coincide con cualquier espacio en blanco (espacio, tabulador, nueva línea).
  • \S: Coincide con cualquier carácter que no sea un espacio en blanco.
  • ^: Coincide con el inicio de la cadena.
  • $: Coincide con el final de la cadena.
  • +: Coincide con uno o más de los elementos anteriores.
  • *: Coincide con cero o más de los elementos anteriores.
  • {n}: Coincide con exactamente n repeticiones del elemento anterior.
  • {n,m}: Coincide con entre n y m repeticiones del elemento anterior.
  • [abc]: Coincide con cualquier carácter dentro de los corchetes.
  • [^abc]: Coincide con cualquier carácter que no esté dentro de los corchetes.
  • (...): Agrupa una expresión para aplicar operadores a todo el grupo.
