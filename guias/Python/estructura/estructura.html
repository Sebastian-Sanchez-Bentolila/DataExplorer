Estructuras de Datos en Python

Las estructuras de datos son una forma de organizar y almacenar datos de manera que se puedan acceder y modificar eficientemente. Python ofrece varias estructuras de datos integradas que son fundamentales para escribir programas eficientes y organizados.

Listas

Las listas son estructuras de datos ordenadas y mutables que pueden contener elementos de diferentes tipos. Se definen utilizando corchetes [] y se pueden modificar después de su creación.

Creación y Acceso a Listas


# Crear una lista
mi_lista = [1, 2, 3, "a", "b", "c"]

# Acceder a elementos de la lista
print(mi_lista[0])  # Primer elemento
print(mi_lista[-1]) # Último elemento

# Modificar un elemento de la lista
mi_lista[0] = 10
print(mi_lista)

Salida en consola:


1
c
[10, 2, 3, 'a', 'b', 'c']

Métodos de Listas


# Agregar elementos
mi_lista.append(4)
print(mi_lista)

# Eliminar elementos
mi_lista.remove("a")
print(mi_lista)

# Obtener la longitud de la lista
print(len(mi_lista))

Salida en consola:


[10, 2, 3, 'a', 'b', 'c', 4]
[10, 2, 3, 'b', 'c', 4]
6

Tuplas

Las tuplas son estructuras de datos ordenadas e inmutables. Una vez creadas, sus elementos no pueden ser modificados. Se definen utilizando paréntesis ().

Creación y Acceso a Tuplas


# Crear una tupla
mi_tupla = (1, 2, 3, "a", "b", "c")

# Acceder a elementos de la tupla
print(mi_tupla[0])  # Primer elemento
print(mi_tupla[-1]) # Último elemento

# Intentar modificar un elemento (generará un error)
mi_tupla[0] = 10

Salida en consola:


1
c
TypeError: 'tuple' object does not support item assignment

Conjuntos

Los conjuntos son estructuras de datos no ordenadas y mutables que no permiten elementos duplicados. Se definen utilizando llaves {} o la función set().

Creación y Operaciones con Conjuntos


# Crear un conjunto
mi_conjunto = {1, 2, 3, 4, 4, 5}
print(mi_conjunto)

# Agregar elementos
mi_conjunto.add(6)
print(mi_conjunto)

# Eliminar elementos
mi_conjunto.remove(1)
print(mi_conjunto)

# Operaciones de conjuntos
otro_conjunto = {3, 4, 5, 6, 7}
union = mi_conjunto.union(otro_conjunto)
interseccion = mi_conjunto.intersection(otro_conjunto)
print(union)
print(interseccion)

Salida en consola:


{1, 2, 3, 4, 5}
{2, 3, 4, 5, 6}
{2, 3, 4, 5, 6, 7}
{3, 4, 5, 6}

Diccionarios

Los diccionarios son estructuras de datos no ordenadas y mutables que almacenan pares clave-valor. Se definen utilizando llaves {} y cada elemento está compuesto por una clave única y su valor asociado.

Creación y Acceso a Diccionarios


# Crear un diccionario
mi_diccionario = {"nombre": "Ana", "edad": 30, "ciudad": "Buenos Aires"}

# Acceder a valores por su clave
print(mi_diccionario["nombre"])
print(mi_diccionario["edad"])

# Modificar un valor
mi_diccionario["edad"] = 31
print(mi_diccionario)

Salida en consola:


Ana
30
{'nombre': 'Ana', 'edad': 31, 'ciudad': 'Buenos Aires'}

Métodos de Diccionarios


# Agregar un nuevo par clave-valor
mi_diccionario["profesión"] = "Ingeniera"
print(mi_diccionario)

# Eliminar un par clave-valor
del mi_diccionario["ciudad"]
print(mi_diccionario)

# Obtener todas las claves y valores
claves = mi_diccionario.keys()
valores = mi_diccionario.values()
print(claves)
print(valores)

Salida en consola:


{'nombre': 'Ana', 'edad': 31, 'ciudad': 'Buenos Aires', 'profesión': 'Ingeniera'}
{'nombre': 'Ana', 'edad': 31, 'profesión': 'Ingeniera'}
dict_keys(['nombre', 'edad', 'profesión'])
dict_values(['Ana', 31, 'Ingeniera'])

Colas (Queue) y Pilas (Stack)

Las colas y pilas son estructuras de datos que permiten manejar datos siguiendo el principio FIFO (First In First Out) y LIFO (Last In First Out), respectivamente. Python proporciona el módulo collections para trabajar con estas estructuras.

Colas


from collections import deque

# Crear una cola
cola = deque()

# Agregar elementos a la cola
cola.append("a")
cola.append("b")
cola.append("c")
print(cola)

# Eliminar elementos de la cola
elemento = cola.popleft()
print(elemento)
print(cola)

Salida en consola:


deque(['a', 'b', 'c'])
a
deque(['b', 'c'])

Pilas


# Crear una pila
pila = []

# Agregar elementos a la pila
pila.append("a")
pila.append("b")
pila.append("c")
print(pila)

# Eliminar elementos de la pila
elemento = pila.pop()
print(elemento)
print(pila)

Salida en consola:


['a', 'b', 'c']
c
['a', 'b']
