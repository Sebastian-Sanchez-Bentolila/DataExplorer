Módulos en Python

Un módulo en Python es un archivo que contiene definiciones y declaraciones de Python, como funciones, clases y variables, que se pueden reutilizar en otros programas. Los módulos ayudan a organizar y estructurar el código, haciendo que sea más fácil de mantener y reutilizar.

Creación de un Módulo

Para crear un módulo en Python, simplemente escribe tus definiciones y declaraciones en un archivo con extensión .py. Por ejemplo, vamos a crear un módulo llamado mi_modulo.py:


# Archivo: mi_modulo.py
    
def saludar(nombre):
    return f"Hola, {nombre}!"
    
def sumar(a, b):
    return a + b

Importación de Módulos

Para usar las funciones y variables de un módulo en otro archivo, puedes importarlo utilizando la palabra clave import. Aquí tienes un ejemplo de cómo importar y utilizar nuestro módulo mi_modulo.py:


# Archivo: main.py
    
import mi_modulo
    
nombre = "Mundo"
print(mi_modulo.saludar(nombre))
    
resultado = mi_modulo.sumar(3, 5)
print(f"La suma es: {resultado}")

Salida en consola:


Hola, Mundo!
La suma es: 8

Importación de Funciones y Variables Específicas

También puedes importar funciones o variables específicas de un módulo utilizando la sintaxis from ... import .... Aquí tienes un ejemplo:


# Archivo: main.py
    
from mi_modulo import saludar, sumar
    
nombre = "Mundo"
print(saludar(nombre))
    
resultado = sumar(3, 5)
print(f"La suma es: {resultado}")

Salida en consola:


Hola, Mundo!
La suma es: 8

Uso de Alias

Para facilitar el uso de módulos, puedes asignarles alias utilizando la palabra clave as. Esto es útil cuando el nombre del módulo es largo o poco descriptivo. Aquí tienes un ejemplo:


# Archivo: main.py
    
import mi_modulo as mm
    
nombre = "Mundo"
print(mm.saludar(nombre))
    
resultado = mm.sumar(3, 5)
print(f"La suma es: {resultado}")

Salida en consola:


Hola, Mundo!
La suma es: 8

Módulos Integrados

Python incluye una amplia variedad de módulos integrados que proporcionan funcionalidades adicionales. Algunos ejemplos comunes son:

  • math: Proporciona funciones matemáticas.
  • os: Permite interactuar con el sistema operativo.
  • datetime: Proporciona clases para manipular fechas y horas.
  • random: Proporciona funciones para generar números aleatorios.

Ejemplo de Uso del Módulo math


import math
    
# Calcular la raíz cuadrada de 16
resultado = math.sqrt(16)
print(f"La raíz cuadrada de 16 es: {resultado}")

Salida en consola:


La raíz cuadrada de 16 es: 4.0

Creación de Paquetes

Un paquete en Python es una colección de módulos organizados en un directorio que contiene un archivo especial __init__.py. Este archivo puede estar vacío o puede incluir código de inicialización para el paquete.

Estructura de un Paquete

La estructura de un paquete puede verse así:


mi_paquete/
    __init__.py
    modulo1.py
    modulo2.py

Uso de un Paquete

Para utilizar un paquete, simplemente impórtalo utilizando la sintaxis de punto. Aquí tienes un ejemplo:


# Archivo: main.py
    
from mi_paquete import modulo1, modulo2
    
modulo1.funcion1()
modulo2.funcion2()

PIP

pip es el sistema de gestión de paquetes oficial para Python. Permite instalar, actualizar y gestionar paquetes y bibliotecas que amplían la funcionalidad del lenguaje. Es una herramienta fundamental para cualquier desarrollador o analista que trabaje con Python.

¿Qué es pip?

pip es un acrónimo de "Pip Installs Packages". Su objetivo principal es facilitar la instalación y administración de bibliotecas desde el índice de paquetes de Python, conocido como PyPI.

Instalación de pip

En la mayoría de las distribuciones modernas de Python, pip ya está instalado. Sin embargo, si necesitas instalarlo manualmente, puedes usar el siguiente comando: 


python -m ensurepip --upgrade

Esto instalará pip si no está presente o lo actualizará a la última versión disponible.

Comandos básicos de pip

1. Verificar la versión instalada

Usa el siguiente comando para verificar qué versión de pip tienes instalada:


pip --version

Salida en consola:


pip 23.0.1 from /usr/local/lib/python3.9/site-packages/pip (python 3.9)

2. Instalar un paquete

Para instalar un paquete, usa el siguiente comando:


pip install nombre_paquete

Por ejemplo, para instalar la biblioteca numpy, puedes ejecutar: 


pip install numpy

3. Actualizar un paquete

Para actualizar un paquete existente a su última versión:


pip install --upgrade nombre_paquete

4. Listar paquetes instalados

Obtén una lista de todos los paquetes instalados y sus versiones:


pip list

5. Desinstalar un paquete

Si deseas eliminar un paquete de tu entorno:


pip uninstall nombre_paquete

Uso avanzado de pip

Instalación desde un archivo requirements.txt

Un archivo requirements.txt es una lista de dependencias necesarias para un proyecto. Puedes instalar todas las bibliotecas mencionadas en este archivo con: 


pip install -r requirements.txt
Nota: Este comando es útil en proyectos colaborativos, ya que asegura que todos los desarrolladores trabajen con las mismas versiones de las dependencias.

Instalación desde un repositorio

pip también permite instalar paquetes directamente desde un repositorio Git:

pip install git+https://github.com/usuario/repositorio.git

Ejemplo práctico

Vamos a crear un entorno de desarrollo virtual, instalar un paquete y comprobar que funciona: 


# Crear un entorno virtual
python -m venv mi_entorno

# Activar el entorno virtual
# En Windows:
mi_entorno\Scripts\activate
# En Mac/Linux:
source mi_entorno/bin/activate

# Instalar numpy
pip install numpy

# Comprobar instalación
python -c "import numpy; print(numpy.__version__)"

Salida en consola:


1.23.5
