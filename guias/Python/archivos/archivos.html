Manejo de Archivos y el Módulo os en Python

El manejo de archivos es una habilidad fundamental en la programación. Python proporciona varias funciones y métodos para trabajar con archivos. Además, el módulo os ofrece herramientas para interactuar con el sistema operativo. Vamos a explorar estos temas en detalle.

Abrir y Cerrar Archivos

Para abrir un archivo en Python, usamos la función open(). Esta función retorna un objeto de archivo, que podemos usar para leer, escribir o realizar otras operaciones en el archivo. Aquí un ejemplo básico:


archivo = open('mi_archivo.txt', 'r')
contenido = archivo.read()
print(contenido)
archivo.close()

Salida en consola:


Contenido del archivo...

Es importante cerrar el archivo después de usarlo para liberar recursos. También podemos usar la declaración with para manejar archivos, lo que asegura que el archivo se cierre automáticamente:


with open('mi_archivo.txt', 'r') as archivo:
    contenido = archivo.read()
    print(contenido)

Modos de Apertura de Archivos

La función open() acepta un segundo argumento que especifica el modo en que queremos abrir el archivo:

  • 'r': Lectura (por defecto, el archivo debe existir).
  • 'w': Escritura (crea un archivo nuevo o sobrescribe uno existente).
  • 'a': Anexar (añade contenido al final del archivo).
  • 'b': Binario (se usa con otros modos, p.ej., 'rb' o 'wb').

Leer y Escribir Archivos

Existen varios métodos para leer y escribir en archivos:

Leyendo Archivos

  • read(): Lee todo el contenido del archivo.
  • readline(): Lee una línea del archivo.
  • readlines(): Lee todas las líneas y las retorna como una lista.

with open('mi_archivo.txt', 'r') as archivo:
    for linea in archivo:
        print(linea.strip())

Escribiendo en Archivos

  • write(): Escribe una cadena al archivo.
  • writelines(): Escribe una lista de cadenas al archivo.

with open('mi_archivo.txt', 'w') as archivo:
    archivo.write("Esto es una línea nueva")

Uso del Módulo os

El módulo os nos permite interactuar con el sistema operativo de una manera portátil. Algunas operaciones comunes incluyen:

  • Renombrar un archivo:

import os
os.rename('mi_archivo.txt', 'archivo_renombrado.txt')
  • Eliminar un archivo:

import os
os.remove('archivo_renombrado.txt')

Explorando Directorios

Podemos usar os para trabajar con directorios. Aquí algunos ejemplos:

  • Obtener la lista de archivos en un directorio:

import os
archivos = os.listdir('.')
print(archivos)
  • Crear un nuevo directorio:

import os
os.mkdir('nuevo_directorio')
  • Cambiar el directorio actual:

import os
os.chdir('nuevo_directorio')
print(os.getcwd())

Manejo de Rutas de Archivos

El módulo os también proporciona funciones útiles para manipular rutas de archivos:

  • Unir directorios y nombres de archivos:

import os
ruta_completa = os.path.join('directorio', 'archivo.txt')
print(ruta_completa)
  • Obtener la ruta absoluta:

import os
ruta_absoluta = os.path.abspath('archivo.txt')
print(ruta_absoluta)
  • Comprobar si una ruta es un archivo o un directorio:

import os
es_archivo = os.path.isfile('archivo.txt')
es_directorio = os.path.isdir('directorio')
print(es_archivo, es_directorio)

Notas y Advertencias

Nota: Al manipular archivos y directorios, ten cuidado de no borrar o sobrescribir archivos importantes accidentalmente.
