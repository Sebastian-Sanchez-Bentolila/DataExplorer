Números Complejos: Teoría y Aplicaciones

Los números complejos amplían el conjunto de los números reales, permitiendo la existencia de soluciones para ecuaciones como \( x^2 + 1 = 0 \). Se definen como:

\[ z = a + bi \]

donde \( a \) es la parte real, \( b \) la parte imaginaria, y \( i \) es la unidad imaginaria, que cumple:

\[ i^2 = -1 \]

Representación de los Números Complejos

Los números complejos pueden representarse de varias maneras:

  • Forma binómica: \( z = a + bi \)
  • Forma polar: \( z = r(\cos \theta + i \sin \theta) \), donde \( r = |z| \) y \( \theta = \arg(z) \)
  • Forma exponencial: \( z = re^{i\theta} \), usando la identidad de Euler

Módulo y Argumento

El módulo de un número complejo \( z = a + bi \) es:

\[ |z| = \sqrt{a^2 + b^2} \]

El argumento (ángulo) es:

\[ \theta = \tan^{-1} \left(\frac{b}{a}\right) \]

Ejemplo

Para \( z = 3 + 4i \):

  • Módulo: \( |z| = \sqrt{3^2 + 4^2} = 5 \)
  • Argumento: \( \theta = \tan^{-1} \left(\frac{4}{3} \right) \approx 53.13^\circ \)

Operaciones con Números Complejos

  • Suma y resta: Se operan las partes reales e imaginarias por separado.
  • Multiplicación: Se usa la propiedad \( i^2 = -1 \).
  • División: Se multiplica por el conjugado del denominador.

Ejemplo

\[ (3 + 4i) + (2 - i) = (3+2) + (4-1)i = 5 + 3i \]

Potencias y Raíces

Para calcular potencias, usamos la fórmula de De Moivre:

\[ z^n = r^n (\cos n\theta + i\sin n\theta) \]

Las raíces \( n \)-ésimas se encuentran como:

\[ z_k = \sqrt[n]{r} \left( \cos \frac{\theta + 2k\pi}{n} + i \sin \frac{\theta + 2k\pi}{n} \right) \]

Ejemplo

Las raíces cúbicas de \( 8i \) se obtienen con \( r = 8 \) y \( \theta = 90^\circ \):

  • \( z_0 = 2 (\cos 30^\circ + i\sin 30^\circ) \)
  • \( z_1 = 2 (\cos 150^\circ + i\sin 150^\circ) \)
  • \( z_2 = 2 (\cos 270^\circ + i\sin 270^\circ) \)

Números Complejos en Python y R

Python

Python tiene una librería nativa para números complejos:


# Definir un número complejo
z = complex(3, 4)

# Operaciones básicas
print("Parte real:", z.real)
print("Parte imaginaria:", z.imag)
print("Módulo:", abs(z))
print("Conjugado:", z.conjugate())

# Graficar números complejos
import matplotlib.pyplot as plt

def graficar_complejo(z):
    plt.figure(figsize=(5,5))
    plt.axhline(0, color='black', lw=1)
    plt.axvline(0, color='black', lw=1)
    plt.grid(True, linestyle='--', alpha=0.6)
            
    plt.quiver(0, 0, z.real, z.imag, angles='xy', scale_units='xy', scale=1, color='red')
    plt.xlim(-10, 10)
    plt.ylim(-10, 10)
    plt.xlabel('Re')
    plt.ylabel('Im')
    plt.title(f'Número complejo {z}')
    plt.show()

# Ejemplo de graficado
graficar_complejo(z)

Salida en consola: 


Parte real: 3.0
Parte imaginaria: 4.0
Módulo: 5.0
Conjugado: (3-4j)

R

En R, los números complejos se manejan de la siguiente manera:


# Definir un número complejo
z <- complex(real = 3, imaginary = 4)

# Operaciones básicas
Re(z)  # Parte real
Im(z)  # Parte imaginaria
Mod(z) # Módulo
Conj(z) # Conjugado

# Graficar en el plano complejo
library(ggplot2)

graficar_complejo <- function(z) {
df <- data.frame(x = c(0, Re(z)), y = c(0, Im(z)))
        
ggplot(df, aes(x, y)) +
    geom_segment(aes(xend = Re(z), yend = Im(z)), arrow = arrow(type="closed", length = unit(0.2, "inches")), color="red") +
    geom_hline(yintercept = 0, linetype="dashed") +
    geom_vline(xintercept = 0, linetype="dashed") +
    theme_minimal() +
    labs(title = paste("Número Complejo:", z), x = "Parte Real", y = "Parte Imaginaria")
}

# Ejemplo de graficado
graficar_complejo(z)

Salida en consola: 


> z <- complex(real = 3, imaginary = 4)
> Re(z)  # Parte real
[1] 3
> Im(z)  # Parte imaginaria
[1] 4
> Mod(z) # Módulo
[1] 5
> Conj(z) # Conjugado
[1] 3-4i

Aplicaciones de los Números Complejos

  • Ingeniería eléctrica: Representación de señales de corriente alterna.
  • Física cuántica: Ecuaciones de onda y mecánica cuántica.
  • Computación gráfica: Transformaciones en el plano complejo.
  • Análisis de ecuaciones diferenciales: Soluciones en sistemas dinámicos.
